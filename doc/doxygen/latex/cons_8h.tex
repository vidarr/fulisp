Accessing cons cells. 

Iterate over list of cons cells and execute code on every single car (eqv. of Common Lisp's \char`\"{}mapcar\char`\"{}). Iterates over the list, assigns first car to run\+Var, then executes \char`\"{}code\char`\"{}, then assigns next car to run\+Var and goes on like this until the end of the list has been reached. \char`\"{}code\char`\"{} can use \char`\"{}run\+Var\char`\"{} to subsequently access every car.
