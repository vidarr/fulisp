Accessing cons cells.

Iterate over list of cons cells and execute code on every single car (eqv. of Common Lisp's "mapcar"). Iterates over the list, assigns first car to runVar, then executes "code", then assigns next car to runVar and goes on like this until the end of the list has been reached. "code" can use "runVar" to subsequently access every car.

Parameters
envcurrent environment
listconsed list
runVarvariable to use as iterator. Must be type "struct Expression *"
codethe code to map onto the list's entries
